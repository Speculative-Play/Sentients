# Sentients 

The centralized app for all Speculative Play AI chatbots. 

## Includes 
1. Chabot 
2. Jay Mort 
3. S.A.M 

## Pre-reqs
1. Node v20
2. Yarn 1.22.22

## Setting it up 
1. Clone repo 
2. Open Terminal in directory 
3. Run `yarn` 
4. Then run `yarn dev` - this should open dev server [here](http://localhost:5173)
